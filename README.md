# AutoCase AI - Automation Testing Platform

AutoCase là nền tảng automation testing low-code/no-code với AI tích hợp, giúp QA:

- Tạo test case từ form schema
- Sinh script Playwright/Selenium tự động
- Tạo workflow từ các test case
- Chạy test và nhận report chi tiết

## Cấu trúc dự án

- backend/: Backend C# (.NET 7) – API, AI integration
- chrome-extension/: Extension để QA pick element và gửi form schema
- ai-prompts/: Thư viện prompt dùng cho AI
- docs/: Tài liệu, thiết kế hệ thống
